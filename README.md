# CSE518: Artificial Intelligence Project

## Overview
This repository contains a comprehensive artificial intelligence project completed as part of the CSE518 course. The project focuses on implementing and comparing heuristic search algorithms with practical applications in decision-making and problem-solving.

## Project Highlights

### Key Components
- **Heuristic Algorithm Implementation** - Development and comparison of multiple AI search heuristics
- **Decision Tree Analysis** - Building and visualizing decision trees for intelligent decision-making
- **Performance Evaluation** - Systematic comparison of algorithm performance across different metrics
- **Data Analysis** - Statistical analysis with visualizations and comparative reports

### Skills Demonstrated
- Algorithm design and implementation
- Data analysis and visualization
- Comparative performance evaluation
- Machine learning and AI problem-solving
- Jupyter Notebook development for reproducible research

## Project Deliverables

| File | Description |
|------|-------------|
| `CSE518_Project.ipynb` | Initial project implementation and exploration |
| `CSE518_Project_Part2.ipynb` | Extended analysis and algorithm development |
| `Comparison_Report.ipynb` | Detailed heuristic algorithm comparison results |
| `Deliverable_3_Descriptive_Report.pdf` | Comprehensive project report with findings |
| `heuristic_comparison_results.csv` | Algorithm performance metrics and results |

## Technologies & Tools
- **Python** - Core language for implementation
- **Jupyter Notebook** - Interactive development and documentation
- **Data Visualization** - Statistical charts and decision tree diagrams
- **Machine Learning** - Algorithm implementation and evaluation

## Key Findings
The project includes comparative analysis of different heuristic approaches, with visualizations demonstrating performance differences and optimal use cases for each algorithm.

## How to Use
1. Clone the repository
2. Open the Jupyter notebooks with Jupyter Lab or Jupyter Notebook
3. Run cells sequentially to execute algorithms and view results
4. Refer to `Deliverable_3_Descriptive_Report.pdf` for detailed findings
